# Lakehouse Architecture Project

This repository contains the projects and practical assignments for the Data Lakehouse architecture course, featuring a modern open-source stack (MinIO, Apache Iceberg, Spark, Airflow, and Superset).

---

Este repositório contém os projetos e entregas práticas do curso de arquitetura de dados Data Lakehouse, utilizando uma stack open-source moderna (MinIO, Apache Iceberg, Spark, Airflow e Superset).
